## 🛠 Using the interface

The main window displays your account list. You add a new profile by pressing the plus button. Enter your credentials into the form that appears. The software saves these details in an encrypted file on your device. To launch a profile, select the desired row in the list and press the play button. 

You manage your fastflags through the settings menu. You enable or disable these custom settings by checking the provided boxes. Changes take effect the next time you launch a game instance. 

## 📦 Managing multiple instances

This tool supports multiple instances. You run several games at once by selecting multiple accounts in the list and clicking launch. You move accounts into groups using the folder icon. Drag and drop profiles to organize your dashboard. 

## 🛡 Security and privacy

The application stores your sessions on your local drive. No data travels to external servers. You hold control over your login cookies. Clear your sessions by pressing the logout button within the application dashboard. If you delete the application, delete the folder containing your data to remove all local traces.

## 📝 Troubleshooting common issues

Review these steps if you experience problems during use.

*   The application does not open: Ensure you installed the latest version of the Net Runtime from Microsoft.
*   Logins fail: Check your internet connection. Some profiles expire after a set time. Refresh the session by clicking the update button.
*   Fastflags conflict: Reset your settings to the default state using the restore button in the options tab. 
*   Missing accounts: Confirm the save path in the settings menu points to the correct folder on your drive.
*   Performance lag: Close other programs while running multiple game instances to save system memory.
